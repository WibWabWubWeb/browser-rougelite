Map Structure & Node Types

How do players choose their path through the sector?

Node Types
  • ⚔️ Skirmish: Standard battle.
  • ☠️ Elite: Tougher battle, better rewards.
  • 🛒 Trade Hub: Shop for units and upgrades.
  • ❓ Distress Signal: Random event (narrative/choice).
  • 🔧 Repair Bay: Heal damaged units.
  • 🛸 Boss: Final sector encounter.
Pathing Logic
The Web (Slay the Spire)

Branching paths that move forward. Can see several nodes ahead.

Sector Map (FTL)

Nodes scattered in space. Must jump forward before the "Fleet" catches you.
